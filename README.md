# Traveloop

Traveloop is a full-stack Next.js + Supabase SaaS platform designed for collaborative travel planning. Built for the modern traveler, it helps users build itineraries, manage budgets, discover activity recommendations using AI (Claude API), and collaborate in real-time with fellow tripmates. 

### Core Features
* **Interactive Itinerary Builder:** Create trips, manage dates, and add specific activities for each destination.
* **Collaborative Planning:** Real-time trip sharing and role-based permissions (viewers, editors, admins) via Supabase Realtime.
* **Smart Recommendations:** AI-powered activity and sightseeing suggestions tailored to your budget and interests.
* **Expense & Budget Tracking:** Monitor trip costs and categorize expenses.
* **Modern Tech Stack:** Built with Next.js 14, React 18, TypeScript, Tailwind CSS, shadcn/ui, and Supabase.
